Manual_Latency_Analysis.R 
- Analyse and plot manual latency data from birds with no neural manipulations

Inj_manual_latency_analysis.R

SLEAP_Analysis.R 
- Evaluate tracking data for all birds

- Run statistics for indivual metrics of birds with no neural manipulations 

Example_plots.R
- Visual representation of movement metrics with example data from 2 birds 

PCA_analysis.R
- Pricipal component analysis of movement metrics over varrying time intervals
